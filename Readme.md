# To-Do List App

A simple and beautiful To-Do List app built with HTML, CSS, and JavaScript.

## Features
- Add tasks
- Edit tasks
- Mark tasks as completed
- Delete single tasks
- Delete all tasks
- Persistent storage with Local Storage

## How to Run
1. Clone the repository
2. Open `index.html` in your browser
